# Whoamifuck

## Ax Introduction
<p>Eonian_Sharp发布的第一款开源工具,这是一款由shell编写的检测入侵用户的工具</p>
<p>这个工具方便了我们查看登入服务器的用户，使用了正则表达式去提取，去重，省去了我们看日志的繁琐。</p>
